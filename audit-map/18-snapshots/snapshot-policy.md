# Snapshot Policy

- Snapshots record environment state, not just narrative conclusions.
- Use `18-snapshots/repo/` for path trees, config snapshots, and git-state notes.
- Use `18-snapshots/live/` for deployment observations.
- Use `18-snapshots/env/` for machine, toolchain, and model-cache state.
- Use `18-snapshots/accounts/` for non-secret account/provider posture summaries.
